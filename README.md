# Four-gene diagnostic signature for Kashin–Beck disease

This repository provides all scripts used in this study.

## Workflow

1. RNA-seq preprocessing (DESeq2)
2. Microarray preprocessing (limma / normalization)
3. Differential expression analysis
4. WGCNA
5. Machine learning model construction

## Notes

- Raw data are available from GEO
- Scripts are organized by analysis step
